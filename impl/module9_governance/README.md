# 模块9：治理筛查 (Governance Screening)

> P0任务 | 数据源：akshare(免费) + 年报推断 | 目标：5分钟内输出结论

## 核心功能

对A股上市公司进行治理风险筛查，输出三项核心指标 + 综合结论：

| 指标 | 数据源 | 阈值 |
|------|--------|------|
| 股权质押比例 | akshare `stock_gpzy_pledge_ratio_em`（东方财富） | <25%正常 / 25-50%偏高 / >50%高危 |
| 审计意见（近5年） | 年报推断 + 已知案例库 | 出现非标意见 → 高风险 |
| 商誉/净资产 | akshare `stock_financial_abstract`（同花顺） | <10%正常 / 10-30%偏高 / >30%高危 |

## 结论判断

- **通过（🟢）**：三项均正常，得分=0
- **存疑（🟡）**：1-2项偏高，或审计推断存疑
- **高风险（🔴）**：单项严重（质押>50%/非标审计/商誉>30%），或命中已知问题公司

## 快速使用

```python
from module9_governance import screen

# 单只股票
report = screen.screen("002014", "永新股份")
print(report.summary())

# 批量（推荐，共享质押数据缓存）
r1, r2 = screen.screen_pair("002014", "永新股份", "600518", "康美药业")
```

## 数据源

| 数据项 | 主要来源 | 降级方案 |
|--------|---------|---------|
| 股权质押 | `stock_gpzy_pledge_ratio_em`（东方财富，全市场） | cninfo质押记录聚合 |
| 股东结构 | `stock_circulate_stock_holder`（THS同花顺） | 手动查找 |
| 审计意见 | 年报财务指标推断 + 已知案例库 | 年报PDF解析（未实现） |
| 商誉/净资产 | `stock_financial_abstract`（THS同花顺） | 年报PDF解析（未实现） |
